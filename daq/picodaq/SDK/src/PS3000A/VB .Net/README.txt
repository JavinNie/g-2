PicoScope 3000 (A API) VB .NET Example
======================================

This is a basic VB .NET Console example that demonstrates how to open a connection to a PicoScope 3000 series oscilloscope using the ps3000a driver.

The example will output information about the device and then close the connection.

Please note how the data types are defined for the function declarations in relation to the PicoScope 3000 Series (A API) PC Oscilloscopes and MSOs Programmer's Guide.
